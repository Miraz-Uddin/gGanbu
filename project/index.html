Squid Game - gGanbu

Game Time: 1minutes, 15seconds

Timer: 0000:00

Rules of this Game
  • Game will start, After clicking the 'Start' Button
  • Game will be abandoned & reset, when Clicking the 'Reset' Button
  • 'player 1' & 'player 2' will give a number as input
  • Suppose it's player 1's turn, so 'player 1' will have to choose if the number given by 'player 2' is Even or Odd
  • If 'player 1' guess correctly the number given by 'player 2' will be added as points of 'player 1' and the number given by 'player 2' will be deducted as points of 'player 2'
  • The above 3 steps will be repeated again, but this time the turn will be for 'player 2'
  • The turns will change after clicking 'submit' button
  • In this way the game will move on, until we found a 'Winner'
  • If any player can't achieve 20 points before the timer reach to Game Time, then the game will be counted as 'DRAW'
